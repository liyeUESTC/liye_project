## 新疆喀什
```
喀什属于南疆，与塔吉克斯坦很近，驱车大概300公里就可以到达边境。
喀什与乌鲁木齐以天山为界。
从成都到喀什，航线需要3300多公里，驾车需要4000多公里。
喀什跟成都有两个小时时差。
```
```
第一天
早上9:30航班成都飞喀什，下午2点40左右到，然后入住全季酒店。
下午去了喀什老城，观看了类似中东风格的建筑和工艺品。
晚上去了喀什美食广场，观看了印象喀什。
```

```
第二天
达瓦昆沙漠，4A级景区，坐了沙漠飓风车，骑了骆驼。吃到了狼坑烤的架子肉，羊肉串，感受了沙漠之光。
下午去看了千年胡杨王。
帕米尔冰泉，源自海拔7509米的慕士塔格峰。确实很好喝。
晚上印象喀什再来一波，喝了乌苏酒，感觉还不错。
```
```
第三天，参观。买了些许新疆特产。踏上回成都的飞机。
```
