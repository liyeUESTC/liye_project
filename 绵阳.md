## 绵阳（梓潼&小垭）
```
18年国庆，背负着任务，踏上了绵阳之旅，路途之所见所闻，分享与大家。
```

### 小垭

```
小垭是柚子之乡，满山遍野的柚子，可谓是柚子王国。
全乡围绕一个十字路口建设，民风淳朴，基本满足生活所需。
但确实比较闲，唯一的遗憾，是没能在小垭小学的篮球场展现自己对篮筐的暴虐。
```
### 梓潼
```
小垭到梓潼有摆渡车，途径徐州。全程大概需要45分钟。
所经公路虽然是双行道，但是弯度和坡度都比较具有挑战，希望我在买车能顺利进化为秋名山车神。
梓潼县相较于神池县，在经济和文化方面后远超神池。房价大概四五千，可能是神池的2倍之多。
梓潼酥饼是比较有名的小吃，但我并没有觉得很好。吃了梓潼的满江红火锅，并不觉得有多么特色。
还跟着蒲老板一同蹭了喜酒，聚源大酒店，饭菜还不错，感觉比神池的档次高2个level。
仙海，距离梓潼县城35公里左右，无需门票。主要是水上项目，玩了一个水上划船，体验还不错。
七曲山大庙，距离梓潼县城7公里左右，40元门票。佛家圣地，魁星楼是求学圣地，香火非常旺盛。
买了最便宜的68的香火，点了蜡，上了香，求了神，拜了佛。
```

### 绵阳
```
返回成都，途径绵阳，小玩2天。绵阳距离成都140公里，驾车需要2小时。
海逸商务酒店，市中心医院附近。价格100多一些，条件一般，但是距离各景点都比较近。对面小吃街，沾沾比较推荐。
马家巷：小吃一条街，矮子馅饼，余记萝卜干，余记锅巴凉粉，芝士热狗，春卷。
马家巷小吃攻略链接：
http://baijiahao.baidu.com/s?id=1583121093778548473&wfr=spider&for=pc&isFailFlag=1&from=singlemessage&isappinstalled=0
一号桥：夜景还不错，有点“世纪大桥”的感觉，可以沿着人行道走到桥的上面。
越王楼：高度99米，是四大楼阁中最高的，夜景很棒。越王楼霸气、黄鹤楼大气，滕王阁才气、岳阳楼秀气。
需要注意的是，绵阳的夜景到了晚上10:30之后，会全部断电。
绵阳干锅也是一绝：六月雪和三之首，只尝试了六月雪，排骨虾干锅中的排骨特别好吃。但据说，六月雪的干锅更棒。
```
